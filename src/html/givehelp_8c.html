void givehelp ( ESTADO  e)

Função que altera o Estado e dá ajudas.

A função inicialmente verifica se o número de ajudas disponivél é maior que 0, se o for, então é criada uma solução de state , sendo então utilizada a função putnatural , de forma a encontrar peças sem verificar a solução. Se putnatural não encontrar nenhuma ajuda então é utilizada a função searchcomp com indc a 0 de forma a que não utilize recursividade. Se não encontrar nenhum elemento a alterar desta forma então corre a função searchcomp com indc a 1 , então será utilizada recursividade e mais de uma peça será alterada.

Faz um procura complementária entre dois estados diferentes.

A função percorre ambos os estados, sendo que comp deve corresponder à solução do estado src. A cada posição que corresponda a uma peça do tipo SOLTO ou VAZIA verifica que está é igual à peça que se encontra na mesma posição na solução, caso sejam iguais as peças nenhuma operação é efetuada. Caso contrário, o elemento em questão é alterada para a peça correspondente na solução. O que gera três casos:

  • A nova peça é válida, então o elemento é alterado, e colocado na STACK .
  • A nova peça é inválida e indc é 0 , então a peça será alterada para o elemento antigo que lá se encontrava.
  • A nova peça é inválida e indc é diferente de 0 , então é utilizada recursividade de forma a alterar um número arbitrário de peças até que o tabuleiro se torne válido.
