void givehelp ( ESTADO  e)

Função que altera o Estado e dá ajudas.

A função inicialmente verifica se o número de ajudas disponivél é maior que 0, se o for, então é criada uma solução de state , sendo então utilizada a função putnatural , de forma a encontrar peças sem verificar a solução. Se putnatural não encontrar nenhuma ajuda então é utilizada a função searchcomp com indc a 0 de forma a que não utilize recursividade. Se não encontrar nenhum elemento a alterar desta forma então corre a função searchcomp com indc a 1 , então será utilizada recursividade e mais de uma peça será alterada.
