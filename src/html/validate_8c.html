Verifica se dado um vetor uma posição é válida no estado. A função segue a trajetória indicada pelo vetor passado, bem como no sentido aposto a este. Utilizando para tal as macros contain e countval para obter a contagem de elementos iguais segundo a direção do vetor passado como argumento.

irepresenta a linha que se pretende verificar.

vectorjrepresenta a componente das colunas do vetor utilizado.
stateEstado do qual se pretende analisar a grelha.
Returns
Quantas peças do mesmo valor há à volta de uma certa posição, segundo a direção do vetor.
